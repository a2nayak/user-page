#### Favorite programming language

For me, this would be Kotlin. Here's some example Kotlin code that computes the $n$th Fibonacci number for some natural $n \in \mathbb{N}$.

```kotlin
fun fib(n: Int): Int {
    if (n <= 1) return n
    return fib(n-1) + fib(n-2)
}
```
