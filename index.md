#### Favorite programming language

For me, this would be [Kotlin](https://kotlinlang.org/). Here's some example Kotlin code that computes the nth Fibonacci number for some natural number n.

```kotlin
fun fib(n: Int): Int {
    if (n <= 1) return n
    return fib(n-1) + fib(n-2)
}
```
